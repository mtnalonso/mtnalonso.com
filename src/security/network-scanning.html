Escaneo de redes

Un escaneo de redes es el proceso destinado a obtener información acerca del objetivo mediante la identificacion de equipos, puertos y servicios dentro de una red.


Tipos de escaneo

ICMP Echo Scan
Consiste en enviar peticiones ICMP ECHO a un equipo determinado. Si dicho equipo está operativo, este devolverá una respuesta ICMP ECHO.

Ping sweep: Técnica para identificar los diferentes equipos dentro de una red mediante el envio de peticiones ICMP ECHO a múltiples equipos.
TCP / Full Open Scan
Stealth Scan
XMAS Scan
ACK flag probe scan
IDLE/IPID Header Scan
UDP Scanning
